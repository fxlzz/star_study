## 文本样式
加粗：**加粗文本** 或 __加粗文本__

斜体：*斜体文本* 或 _斜体文本_

加粗斜体：***加粗斜体***

~~删除线~~：~~删除线内容~~

## 列表 

无序列表
使用 *、+ 或 - 加上空格。
+ 项目一
+ 项目二
  + 子项目 1
  + 子项目 2

有序列表
使用数字加英文句点。
1. 第一步
2. 第二步

**行内代码**

使用三个反引号

```js
console.log("hello, world");
```
