为什么会出现这个东西呢？
在我们平时开发过程中，会依赖其他的第三方依赖，以包的形式提供第三方服务。
为了减少熵增，我们可以使用 *包管理工具* 。

常见的包管理工具：
- `npm`
- `yarn`
- `pnpm`
- ...
现代前端项目一般使用 `pnpm` 来安装依赖
---
`npm install --save-dev` : 安装依赖
- `--save` ：仅安装依赖
- `-dev` ： 仅安装开发依赖

SemVer （语义化版本） 范围操作符：
- **`"1.2.3"` (无符号)：** 锁定版本。永远只用 `1.2.3`
- **`"~1.2.3"`：** 会更新到 `1.2.x` 的最新版（如 `1.2.9`），但**绝不会**升到 `1.3.0`
- **`"^1.2.3"`：** 会更新到 `1.x.x` 的最新版（如 `1.5.2`），但**绝不会**升到 `2.0.0`
