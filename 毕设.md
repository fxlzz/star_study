> 个人知识库： 问答型 AI 应用

技术栈：
+ 前端
	+ 语言： TS  
	+ 框架： React.js  
	+ 状态管理库： Zustand 
	+ 异步状态管理库： Tanstack Query 
	+ UI库： Antd 
+ 后端
	+ 语言： Node.js(express)
	+ REST API
+ 数据库： 
	+ NoSQL数据库： mongo 
	+ 向量数据库： chroma
+ AI相关： langchain + qwen / openai / ollama
+ 代码管理： git
+ 包管理工具： pnpm
+ 部署： docker

功能：
+ 知识问答 - 提取关键术语 -> 返回相关文档、关联概念
+ 支持处理各类文档 - pdf、txt、json、md、html等
+ 标签管理
+ 知识图谱
