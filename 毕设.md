项目名称：`good-idea`
非常简单的 title “好想法”，记录好想法

# 明确目标
个人知识库 AI 助手 -> 面向个人用户（因为采用 B/S 架构，数据不能存在本地磁盘） -> 私有化 -> 部署到个人服务器上

两个注意：
+ 数据所有权
+ 数据安全

运行在用户本机上的后端服务 + 浏览器前端服务

一体化容器部署： 后端、数据库、向量引擎、文件存储打包在一起，用户只需一键启动

# UI 设计
> 采用 figma 设计功能，加上 ai 真的相当强

[figma](https://www.figma.com/make/kpKQcAaid1sBFLnSYTWDNe/good-idea?p=f&t=ROev5ye0RtYr40Va-0)

> UI 类似于 Obsidian 
- 顶部 header 栏功能：
	+ 左侧： click 弹出 modal 操作
		+ 所有文章
		+ 搜索
		+ 书签
	+ 右侧
		+ ai 助手
		+ 用户信息 avatar
- 二级导航 nav 栏功能：
	+ 新增文件
	+ 新增文件夹
	+ 排序
	+ 文件树内容
- 侧边 sidebar 菜单栏功能：
	+ 白板（画图）
	+ 知识图谱
	+ 终端 cli -> 整个系统的基础的功能，支持 git 操作以及部分 linux 功能操作
--- 

菜单栏用户友好体验：
+ 菜单栏默认使用 icon 展示 ，hover 给予提示
+ 侧边栏区域可改变 width 

中部 main 区域：
基础功能：
+ 识别文件后缀名，展示文件内容， 支持 `.docx | .doc | .pdf | .md | .xlsx`
+ click 侧边栏（白板、关系图谱），展示对应功能的内容部分
+ click 侧边栏的 终端cli，在中部 main 区域的下三分之一部分展示

main 区域用户友好体验：
+ 支持 tabs 切换，一个文件就是一个 tab， 导航栏可支持多个 tabs
+ 可支持左右文章分屏

右侧区域：
+ ai 助手聊天窗口

# 技术实现
+ 前端部分
	react.js + vite + ts + antd
+ 数据持久化
	localStorage + Supabase（云端数据库）
+ 后端部分
	node.js
+ ai 部分
	langchain + qwen api + chroma(向量数据库)
+ 代码迭代 
    git + github
